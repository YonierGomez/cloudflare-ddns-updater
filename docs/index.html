☁️ Cloudflare DDNS Updater

Contenedor Docker multi-arquitectura que mantiene tus registros DNS de Cloudflare sincronizados con tu IP pública. Automático, ligero y listo para correr.

Características

🏗️

Multi-Arquitectura

Corre en amd64, arm64 y armv7. Perfecto para servidores, Raspberry Pi o Apple Silicon.

🔄

Actualización automática

Detecta cambios de IP y actualiza tus registros DNS en Cloudflare automáticamente.

📋

Múltiples dominios

Actualiza varios registros DNS a la vez separándolos por comas.

🛡️

Proxy configurable

Activa o desactiva el proxy de Cloudflare por dominio según tu necesidad.

🐍

Python + Alpine

Imagen mínima basada en Python Alpine. Ligera y con releases automáticos.

📦

Auto-Release

Pipeline CI/CD que detecta nuevas versiones de Python y publica releases con changelog.

Inicio rápido

docker run -d \
  --name cloudflare-ddns \
  --restart unless-stopped \
  -e CLOUDFLARE_EMAIL="tu@email.com" \
  -e CLOUDFLARE_API_TOKEN="tu-api-token" \
  -e ZONE_ID="tu-zone-id" \
  -e DNS_RECORD_NAME="ejemplo.com,sub.ejemplo.com" \
  -e SLEEP_INTERVAL=300 \
  -e PROXY_ENABLED=true \
  neytor/cloudflare-ddns-updater:latest

Docker Compose

services:
  cloudflare-ddns:
    image: neytor/cloudflare-ddns-updater:latest
    container_name: cloudflare-ddns
    restart: unless-stopped
    environment:
      CLOUDFLARE_EMAIL: "tu@email.com"
      CLOUDFLARE_API_TOKEN: "tu-api-token"
      ZONE_ID: "tu-zone-id"
      DNS_RECORD_NAME: "ejemplo.com,sub.ejemplo.com"
      SLEEP_INTERVAL: "300"
      PROXY_ENABLED: "true"
